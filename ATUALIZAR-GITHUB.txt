CONTROLE DE CONVIDADOS LEGA — VERSÃO ESTÁVEL PARA O EVENTO

1. No repositório controle-convidados-lega, envie estes arquivos para a RAIZ.
2. Substitua os arquivos com os mesmos nomes.
3. Não envie a pasta por fora e não use nomes com (2), (4), (7) etc.
4. Os arquivos essenciais são:
   index.html
   firebase-config.js
   manifest.json
   service-worker.js
   logo-lega.png
   icon-192.png
   icon-512.png
5. Aguarde o GitHub Pages concluir a publicação.
6. Abra o site e pressione Ctrl+F5 no computador.
7. No iPad, feche totalmente o aplicativo e abra novamente. Se ainda mostrar a versão antiga, abra o endereço no Safari uma vez e depois reabra o app instalado.

Antes do evento:
- Faça login.
- Abra a lista correta.
- Marque uma entrada de teste e desfaça.
- Atualize a página e confirme que a alteração permaneceu.
- Use “Salvar arquivo do evento” para guardar uma cópia de segurança.
